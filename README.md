# 🚀 Startup Funding Dashboard (Python Dash)

## 📊 Project Overview

This project is a **Data Science Group Project** developed entirely using **Python in Jupyter Notebook**.

The objective of this project is to analyze startup funding data and build an **interactive web-based dashboard** to uncover meaningful insights about:

* Funding trends over time
* Top industries and cities
* Investment patterns
* Key business insights

## 🚀 How to Run the Project

### ▶️ Step 1: Download the Project

* Download or clone this repository
* Extract the folder (if ZIP)

---

### ▶️ Step 2: Open Anaconda Prompt

---

### ▶️ Step 3: Navigate to Project Folder

👉 If project is on Desktop:

```bash
cd Desktop
cd "Startup Funding Analysis Dashboard"
```

👉 If project is in Downloads:

```bash
cd Downloads
cd "Startup Funding Analysis Dashboard"
```

👉 If project is in D drive:

```bash
D:
cd "Startup Funding Analysis Dashboard"
```

👉 Or use full path:

```bash
cd "your-full-folder-path"
```

---

### ▶️ Step 4: Convert Notebook to Python File

```bash
jupyter nbconvert --to python "Startup Funding Dashboard.ipynb"
```

---

### ▶️ Step 5: Run the Dashboard

```bash
python "Startup Funding Dashboard.py"
```

---

### ▶️ Step 6: Open in Browser

After running, you will see:

```bash
Dash is running on http://127.0.0.1:8056/
```

👉 Copy and paste this URL into your browser
